# プロジェクト概要

## 目的

鹿児島市西伊敷の歯科医院「井之上歯科」の公式ウェブサイトを作成・公開する。  
地域の患者様に向け、診療情報・アクセス・予約手段を提供する。

## ゴール

- 患者様が電話番号・診療時間・場所をすぐに確認できる
- スマートフォンでも見やすいレスポンシブデザイン
- 無料で維持できる（Cloudflare Pages 無料枠）

## 技術方針

| 項目 | 選択 | 理由 |
|---|---|---|
| 技術 | 純粋なHTML/CSS/JS | 依存関係ゼロ、メンテナンス容易 |
| ホスティング | Cloudflare Pages | 無料、CDN付き、HTTPS自動 |
| バージョン管理 | GitHub | Cloudflare Pages と連携してCI/CD |
| CMS | なし | コンテンツ変更頻度が低いため |

## ページ構成

1. **ヘッダー** — ロゴ・ナビゲーション・電話番号（固定表示）
2. **ヒーロー** — 医院名・住所・電話番号・CTAボタン
3. **診療時間** — 週間表、休診・予約・駐車場の注記
4. **診療内容** — 4つのサービスカード（一般歯科・予防・小児・義歯）
5. **院長紹介** — 写真・ご挨拶・経歴
6. **患者様の声** — 口コミ3件
7. **アクセス** — Google Maps埋め込み・住所・電話
8. **予約CTA** — 電話番号を大きく強調
9. **フッター** — リンク集・著作権
