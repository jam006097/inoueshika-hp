# 井之上歯科 ホームページ

鹿児島市西伊敷の歯科医院「井之上歯科」の公式ウェブサイト。

## 技術スタック

- **HTML/CSS/JS**: シングルファイル構成（`index.html`）。フレームワーク不使用。
- **ホスティング**: Cloudflare Pages（無料枠）。GitHubプッシュで自動デプロイ。
- **フォント**: Google Fonts（Noto Sans JP / Shippori Mincho / Inter）

## 作業時の注意

- `index.html` は1ファイルにCSS/JS/HTMLを全て含む構成。分割不要。
- 画像はSVGプレースホルダー。実際の写真に差し替える場合は `<!-- TODO -->` コメント箇所を参照。
- React/Babelのtweaks-panel依存は除去済み（デザイン時のみの機能）。
- Cloudflare Pagesのビルド設定: ビルドコマンドなし、出力ディレクトリ: `/`（ルート）。
